# Else in C

---

The `else` Statement

Use the `else` statement to specify a block of code to be executed if
the condition is `false`.

Syntax

```c

if (condition) {
    // ... do something (if conditional true)
} else {
    // ... do something else (if conditional false)
}

```

```c

int time = 20;
if (time < 18) {
    printf("Good day.");
} else {
    printf("Good evening.");
}

// Output is "Good evening"

```
